# Crime Record Indexing & Investigation Workflow System

A console-based C++ project built for DSA course.

## Features
- AVL Tree for crime record storage (O log n operations)
- Linked Queue for investigation workflow (FIFO)
- Linked Stack for evidence handling (LIFO)
- Circular Queue for activity logging
- Insertion Sort, Quick Sort, Merge Sort
- Linear Search and Binary Search
- Expression evaluator using stack (infix to postfix)
- Decimal to Binary converter

## How to Run
1. Open DSA_FINAL_PROJECT.sln in Visual Studio
2. Build the project (Ctrl+Shift+B)
3. Run (Ctrl+F5)

## Technologies
- C++ with templates
- ANSI color codes for UI
- Visual Studio 2022
